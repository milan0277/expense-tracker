# expense-tracker
creating a expense tracker web app , tech stack used : mern .

Ai tool : i have used deepseek to made some changes in the base ui .

Clone the Repository
git clone https://github.com/milan0277/expense-tracker.git

Install Dependencies on both client and server
npm i 

run client
cd client
npm start

run server
cd server
npm start
